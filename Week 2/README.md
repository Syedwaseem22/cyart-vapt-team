# Week 2 – Vulnerability Assessment & Penetration Testing (VAPT)

This folder contains all practical work performed during Week 2 of the VAPT lab.  
The activities follow the complete VAPT lifecycle including scanning, reconnaissance, exploitation, post-exploitation, and a capstone PTES-based project.

## Tools Used
- Kali Linux
- Nmap
- OpenVAS
- Nikto
- Metasploit
- sqlmap
- Maltego
- Shodan

## Folder Structure
- Scanning – Vulnerability identification and prioritization
- Recon – OSINT and asset discovery
- Exploitation – Exploit validation
- Post-Exploitation – Privilege escalation and evidence
- Capstone – Full VAPT cycle
- Screenshots – Proof of execution

All testing was conducted in a controlled lab environment for educational purposes only.
